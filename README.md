# JTANK

#### 介绍
JTANK履带车简介：仓鼠的家庭工坊设计的迷你桌面图传遥控车项目，安卓手机app遥控控制，可驾驶，可查看车辆行驶视角，灯光可开启，适合桌面、床上、地板等地形。目前版本为第三版，前面两个版本由于结构问题将停止发布
